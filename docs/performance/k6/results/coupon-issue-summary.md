## 해석

응답 시간은 현재 몇 초를 기준으로 통과/실패를 판단할지 정책이 확정되지 않았으므로 수집만 수행한다.

실제 DB 잔여 수량 차감 여부는 runbook의 DB 검증 SQL로 추가 확인한다.
